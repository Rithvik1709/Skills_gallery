# Classroom Lesson Architect

## Purpose
Create practical lesson plans from a topic, standard, or learning goal.

## Inputs
- Grade level
- Subject or topic
- Time available
- Learning standard or objective
- Student needs or constraints

## Workflow
1. Define measurable learning objectives.
2. Create an opening activity that checks prior knowledge.
3. Design the main explanation and guided practice.
4. Add differentiated tasks for support and extension.
5. Create formative checks and an exit ticket.
6. Build a simple rubric when assessment is needed.

## Outputs
- Lesson plan
- Activity sequence
- Differentiation ideas
- Assessment prompts
- Rubric or checklist
