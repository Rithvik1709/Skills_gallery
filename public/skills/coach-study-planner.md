# Study Coach Planner

## Purpose
Build realistic study plans that combine scheduling, recall, reflection, and progress tracking.

## Inputs
- Subject or exam
- Deadline
- Current skill level
- Weekly availability
- Weak areas

## Workflow
1. Break the goal into milestones.
2. Create a weekly schedule with review time.
3. Add active recall and practice tasks.
4. Include checkpoints for confidence and accuracy.
5. Adjust the plan based on progress.

## Outputs
- Study calendar
- Daily practice tasks
- Review prompts
- Progress tracker
- Adjustment recommendations
