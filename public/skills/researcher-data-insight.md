# Data Insight Narrator

## Purpose
Turn raw findings, charts, and analysis notes into a concise decision-ready narrative.

## Inputs
- Chart descriptions or data tables
- Key metrics
- Audience
- Decision or question being answered

## Workflow
1. Identify the central insight.
2. Separate signal from noise.
3. Explain supporting evidence in plain language.
4. Note caveats, limitations, and possible confounders.
5. Recommend next actions or follow-up analysis.

## Outputs
- Executive summary
- Insight bullets
- Caveats
- Action recommendations
- Follow-up questions
